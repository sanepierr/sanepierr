# Bidirectional Route Management

This Python script allows users to manage waypoints using the BidirectionalRoute class. It provides a simple command-line interface with options to add, insert, and remove waypoints, as well as traverse the waypoints in both forward and backward directions.

## Prerequisites

- Python 3.x
